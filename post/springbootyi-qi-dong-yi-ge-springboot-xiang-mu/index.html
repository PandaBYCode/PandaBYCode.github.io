SpringBoot（一）启动一个springboot项目

2022-11-01 3 min read # SpringBoot

根据官方文档表示，要创建一个springboot项目，先要在maven项目的pom.xml写为：

<?xml version="1.0" encoding="UTF-8"?>
<project xmlns="http://maven.apache.org/POM/4.0.0" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
    xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 https://maven.apache.org/xsd/maven-4.0.0.xsd">
    <modelVersion>4.0.0</modelVersion>

    <groupId>com.example</groupId>
    <artifactId>myproject</artifactId>
    <version>0.0.1-SNAPSHOT</version>

    <parent>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-parent</artifactId>
        <version>2.7.5</version>
    </parent>

    <!-- Additional lines to be added here... -->

</project>

然后引入依赖

<dependencies>
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-web</artifactId>
    </dependency>
</dependencies>

这里的依赖内容涉及到之后的自动装配

书写一个入口主类：

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

/**
* @author PandaCode.W
*/

@SpringBootApplication
public class hello_boot {

    public static void main(String[] args) {
        SpringApplication.run(hello_boot.class,"args");
    }
}

再书写一个controller类：

import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RestController;

/**
* @author PandaCode.W
*/

@RestController
public class helloController {

    @RequestMapping("/hello")
    public String hello(){
        return "boot,hello";
    }
}

运行主类便可以直接打开一个springboot项目。

在springboot项目中，要管理各种操作，比如修改tomcat启动端口等，都只需要在application.properties的文件中配置即可
例如server.port=6666，即将启动端口改为了6666

springboot与传统的ssm项目最大的差别便是不需要各种繁琐的.xml文件，而是直接通过springboot来控制所有的jar包和类。使用更加方便。
不过因为springboot是对于spring的深层封装，所以底层理解更加复杂。
